myFlix API Documentation

Request URL Description Parameters Request Body Example Response
GET /movies Return a list of all movies None None Array of movie objects
GET /movies/:title Return data about a single movie by title title (String) None Movie object
GET /genres/:name Return data about a genre by name name (String) None Genre object
GET /directors/:name Return data about a director by name name (String) None Director object
POST /users Allow new users to register None { "username": "user", "password": "pass", "email": "user@example.com", "birthday": "YYYY-MM-DD" } Message confirming registration
PUT /users/:username Allow users to update their user info username (String) { "username": "newUser", "password": "newPass", "email": "new@example.com", "birthday": "YYYY-MM-DD" } Message confirming update
POST /users/:username/movies/:movieID Allow users to add a movie to their list of favorites username (String), movieID (String) None Message confirming addition
DELETE /users/:username/movies/:movieID Allow users to remove a movie from their list of favorites username (String), movieID (String) None Message confirming removal
DELETE /users/:username Allow existing users to deregister username (String) None Message confirming deregistration
